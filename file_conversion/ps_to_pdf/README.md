# PostScript to PDF

PostScript (.ps) to PDF converter using Ghostscript.

## Project structure

```
.
└── ps_to_pdf  
    ├── README.md  
    ├── example  
    │   └── 2003-069.ps  
    ├── app.py  
    └── requirements.txt
```

## Install dependencies

```
python3 -m venv .venv
python3 -m pip install -r requirements.txt
pip install -r requirements.txt
```

## Usage

```
python3 app.py input.ps output.pdf
```
